# ⚡ SUPER QUICK START

## 🎯 What This Does
Generates a **RED WALLET-STYLE EMERGENCY CARD** directly in your dashboard!

### Before (Old Way):
Profile → Save → QR Code → Scan QR → See Card ❌

### Now (New Way):
Profile → Save → **CARD APPEARS INSTANTLY!** ✅

---

## 🚀 3-Step Setup

### 1. Install Backend
```bash
cd backend
pip install -r requirements.txt
```

### 2. Start Backend
```bash
python main.py
```
Backend: `http://localhost:8000`

### 3. Start Frontend (New Terminal)
```bash
cd frontend
python -m http.server 3000
```
Frontend: `http://localhost:3000`

---

## 📱 First Time Use

1. **Open:** `http://localhost:3000`
2. **Register:** Create your account
3. **Profile:**
   - Fill in age, gender, **BLOOD GROUP** (required!)
   - Add allergies, conditions, medications
   - Click "Save Profile & Generate Card"
4. **Contacts:**
   - Add emergency contact (name, relation, phone)
   - Set priority
5. **Dashboard:**
   - **YOUR RED CARD APPEARS HERE!** 🎴
   - Click "Download Card" to save as image

## 🐛 Troubleshooting

**No card showing?**
→ Complete profile first, especially blood group!

**Download not working?**
→ Reload page, card should be visible first

**Can't save profile?**
→ Blood group is required!
